# AiCanTest

(TESTED UBUNTU 19.10)

- Requirements:
  - Backend:
    - docker (tested: v19.03.12)
    - docker-compose (tested: v1.26.0)
  - Frontend:
    - node (tested: v12.18.3)
    - npm (tested: v.14.6)
    
- Run Server:
  - Go to root folder
  - sudo docker-compose up


- Run UI:
  - Go to /frontend
  - npm install
  - npm start dev (development) | npm start prod (production)
